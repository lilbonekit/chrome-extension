Welcome!

This extension can:

  • Handle clicks in the extension menu
  • Pin/Unpin the current tab (Ctrl+Shift+Up)
  • Duplicate the current tab (Ctrl+Shift+Down)
  • Move the current tab to the beginning (Ctrl+Shift+Left)
  • Move the current tab to the end (Ctrl+Shift+Right)
  • Show the clock on all pages
  • Change the background of the current page
  • Create reminders
